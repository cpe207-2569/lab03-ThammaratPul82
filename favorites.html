The Standard

เป็นเว็บไซต์ที่เกี่ยวกับการนำเสนอข่าวสาร บทวิเคราะห์ วิดีโอ และพอดแคสต์ที่หลากหลาย ครอบคลุมทั้งประเด็นการเมือง เศรษฐกิจ สังคม และไลฟ์สไตล์ ด้วยเป้าหมายหลักคือการขับเคลื่อนสังคมและสร้างความเปลี่ยนแปลงเชิงบวก

Boostrap

Front-end Framework (ชุดเครื่องมือสำเร็จรูป) ยอดนิยมแบบ Open-source สำหรับพัฒนาเว็บไซต์ ประกอบด้วยชุดคำสั่ง HTML, CSS และ JavaScript ช่วยให้สร้างเว็บไซต์และเว็บแอปพลิเคชันที่สวยงามได้อย่างรวดเร็ว โดยเฉพาะการทำ Responsive Web Design ที่หน้าจอแสดงผลได้ดีทั้งบนมือถือ แท็บเล็ต และคอมพิวเตอร์

Set
